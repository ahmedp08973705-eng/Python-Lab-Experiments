# Aim
To implement various string operations in Python.

# Algorithm
1-Start the program.
2- Declare a string variable.
3-Display the length of the string using len().
4-Convert the string to uppercase using upper().
5- Convert the string to lowercase using lower().
6- Extract a substring using slicing.
7- Replace a word in the string using replace().
7- Display the results.
8- End the program.

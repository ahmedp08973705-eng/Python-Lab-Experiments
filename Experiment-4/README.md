# Aim
To demonstrate list and related functions in Python.

# Algorithm
1- Start the program.
2- Create a list with some elements.
3- Add a new element using append().
4- Remove an element using remove().
5- Display the list elements.
6- Find the length of the list using len().
7- Find the maximum and minimum values using max() and min().
8- Display the results.
9- End the program.

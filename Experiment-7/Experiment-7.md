# Aim
- To demonstrate various data types and operators in Python.

# Algorithm
- Start the program.
- Declare variables of different data types such as integer, float, string, and boolean.
- Display the values of the variables.
- Perform arithmetic operations like addition, subtraction, multiplication, and division.
- Perform comparison operations like greater than, less than, and equal to.
- Display the results.
- End the program.
